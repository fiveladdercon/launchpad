sc_reattach
===========


Prototype
---------

```
$node->sc_reattach()
```


Parameters
----------

_None_

Return Type
-----------

_None_


Description
-----------

**sc_reattach** reinsertes a detached node back into it's parent.

See [sc_detach] for details on detaching a node.

Reattaching a node that is not in the detached state has no effect.
