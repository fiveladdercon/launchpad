uvm.pl
===========

[uvm.pl] outputs the model as a hierarcy of UVM register model instances 
suitable for use in UVM simulations.


Usage
-----

```
spacecraft ... uvm.pl
```


Example
-------

Generate a UVM register model.

```
$ spacecraft ... uvm.pl
```
